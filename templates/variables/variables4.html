In-Place Operators

Ok moving on from input. Let’s talk in-place operators. In-place operators let you write code like 'x = x + 3' more concisely, as 'x += 3'. And who doesn’t love to save time! These operators aren’t just useful for numbers. Check it out:

x = 2
print(x)
2
x += 3
print(x)
5
y = "hello"
print(y)
hello
y += world
print(y)
helloworld

We can do the same thing possible with other operators like -, *, /, % as well.
