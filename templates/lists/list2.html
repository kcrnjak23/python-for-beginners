Strings as Lists

Strings can be indexed like lists too! Indexing a string is like creating a list containing each character in the string. For example:

str = "Hello world!"
print(str[3])
l
print(str[10])
d

Space (" ") is also a symbol and has an index.Space (" ") is also a symbol and has an index.

Trying to access a non-existing index will result in an error.

What's the output of this code?

x = "Python"
print(x[1] + x[4])
