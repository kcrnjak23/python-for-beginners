List Operations

To check if an item is in a particular list, we can use the in operator. It returns True if the item occurs one or more times in the list, and False if it doesn't. Similarly, to check if an item is not in a list, you can use the not operator in one of the following ways:

words = ["spam", "egg", "hello"]
print("spam" in words)
True
print("tomato" in words)
False
print(not "egg" in words)
False
print("world" not in words)
True

The in operator is also used to determine whether or not a string is a substring of another string.
