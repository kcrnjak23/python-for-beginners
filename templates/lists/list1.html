Lists

At their simplest, Lists are used to store items. We can create a list by using square brackets with comas separating items. If you want to access a certain item in the list, you can do this by using its index in square brackets. In our example, that would look like this:

words = ["Hello", "world", "!"]
print(words[0])
print(words[1])
print(words[2])
Hello
world
!

The first list item's index is 0, rather than 1, as you might expect.
